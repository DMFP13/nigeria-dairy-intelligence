# Nigeria Dairy Intelligence

A clean product workspace for building a Nigeria-focused dairy intelligence application.

## Platform Layers

- **Market Signals**: Tracks milk prices, feed costs, weather risk, and supply-demand signals.
- **Farm Economics**: Models farm-level costs, margins, cash flow, and profitability drivers.
- **Herd Intelligence**: Surfaces herd performance, health indicators, and operational anomalies.
- **Scenario Lab**: Tests what-if strategies across pricing, productivity, and investment decisions.
